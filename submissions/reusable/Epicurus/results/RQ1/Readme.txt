This folder contains the scripts to get the results of RQ1 shown in the evaluation.
- RQ1.m: computes the percentage of requirements in which IFBT-UR and UR identifies a v-safe assumption. It also returns The time required for IFBT-UR and UR to identify a v-safe assumption respectively
- comparison_of_TCgen_policies.m: generates the plot in Figure4. Please note that the information indexes are saved in the text files named by the policy name (UR.txt, ART.txt, IFBT_UR.txt, IFBT_ART.txt).
